# 常见问题
* * *
## 账号问题[​](https://api-docs.deepseek.com/zh-cn/faq#%E8%B4%A6%E5%8F%B7%E9%97%AE%E9%A2%98 "账号问题的直接链接")
### 账号无法登录[​](https://api-docs.deepseek.com/zh-cn/faq#%E8%B4%A6%E5%8F%B7%E6%97%A0%E6%B3%95%E7%99%BB%E5%BD%95 "账号无法登录的直接链接")
您账号近期的行为可能触发了我们的自动化风控策略，导致我们暂时关闭了您对账号的访问权限。如需申诉，请填写[问卷](https://trtgsjkv6r.feishu.cn/share/base/form/shrcnDOeld9kcILX57EdfRT8iDg)，我们会尽快处理。
### 邮箱无法注册[​](https://api-docs.deepseek.com/zh-cn/faq#%E9%82%AE%E7%AE%B1%E6%97%A0%E6%B3%95%E6%B3%A8%E5%86%8C "邮箱无法注册的直接链接")
如果您在注册时，收到错误提示“注册失败，暂不支持该邮箱域名注册。”，这是因为您的邮箱不被 DeepSeek 支持，请切换邮箱服务提供商。如仍有问题，请联系 service@deepseek.com。
* * *
## 企业认证[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%BC%81%E4%B8%9A%E8%AE%A4%E8%AF%81 "企业认证的直接链接")
### 个人实名认证与企业实名认证有什么区别？[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%B8%AA%E4%BA%BA%E5%AE%9E%E5%90%8D%E8%AE%A4%E8%AF%81%E4%B8%8E%E4%BC%81%E4%B8%9A%E5%AE%9E%E5%90%8D%E8%AE%A4%E8%AF%81%E6%9C%89%E4%BB%80%E4%B9%88%E5%8C%BA%E5%88%AB "个人实名认证与企业实名认证有什么区别？的直接链接")
个人实名认证账号与企业实名认证账号在用户权益和产品功能上目前没有区别，但认证方式和所需材料有所不同。根据合规要求，请以您账号的实际使用情况进行认证。
### 企业实名账号可以更改为个人账号吗？[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%BC%81%E4%B8%9A%E5%AE%9E%E5%90%8D%E8%B4%A6%E5%8F%B7%E5%8F%AF%E4%BB%A5%E6%9B%B4%E6%94%B9%E4%B8%BA%E4%B8%AA%E4%BA%BA%E8%B4%A6%E5%8F%B7%E5%90%97 "企业实名账号可以更改为个人账号吗？的直接链接")
企业实名认证账号不可变更为个人认证或其他企业。

## 财务问题[​](https://api-docs.deepseek.com/zh-cn/faq#%E8%B4%A2%E5%8A%A1%E9%97%AE%E9%A2%98 "财务问题的直接链接")
### 如何充值？[​](https://api-docs.deepseek.com/zh-cn/faq#%E5%A6%82%E4%BD%95%E5%85%85%E5%80%BC "如何充值？的直接链接")
  * 在线充值：完成实名认证后，您可以在充值页使用支付宝/微信进行在线充值。您可以在账单页查询充值结果。
  * 对公汇款：对公汇款仅支持企业用户。完成企业实名认证后，可获取专属汇款账号，向专属汇款账号进行打款。为保证汇款顺利进行，**请确保汇款方开户名称与开放平台实名认证名称一致** 。我方银行账户到账后，汇款金额将在 10 分钟- 1 小时左右自动转入您的开放平台账户，如未及时收到，请联系我们。


### 余额是否会过期？[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%BD%99%E9%A2%9D%E6%98%AF%E5%90%A6%E4%BC%9A%E8%BF%87%E6%9C%9F "余额是否会过期？的直接链接")
充值余额不会失效过期。赠送余额的有效期您可以在[账单页](https://platform.deepseek.com/transactions)查看。
### 如何申请发票？[​](https://api-docs.deepseek.com/zh-cn/faq#%E5%A6%82%E4%BD%95%E7%94%B3%E8%AF%B7%E5%8F%91%E7%A5%A8 "如何申请发票？的直接链接")
请访问[账单](https://platform.deepseek.com/transactions)页，点击发票管理，申请发票。企业用户开具发票时，发票抬头需要与实名认证信息一致，目前开发票的周期为 7 个工作日左右。
### 充值余额金额不对？[​](https://api-docs.deepseek.com/zh-cn/faq#%E5%85%85%E5%80%BC%E4%BD%99%E9%A2%9D%E9%87%91%E9%A2%9D%E4%B8%8D%E5%AF%B9 "充值余额金额不对？的直接链接")
请再次确认您的充值账户与当前登录的账户一致。
如您有 Google 账号，可尝试进行 Google 登录方式登录开放平台，检查历史充值金额是否在该账号当中。
如果是手机号注册账号，用户操作注销后又重新注册，则新账号与原账号为无关联的独立账号，原账号如有充值无法使用，请[联系我们](https://trtgsjkv6r.feishu.cn/share/base/form/shrcn1lIYKRCKQJrqFjuZCQCdch)。

## API 调用问题[​](https://api-docs.deepseek.com/zh-cn/faq#api-%E8%B0%83%E7%94%A8%E9%97%AE%E9%A2%98 "API 调用问题的直接链接")
### 调用模型时的并发限制是多少？是否可以提高账号的并发上限？[​](https://api-docs.deepseek.com/zh-cn/faq#%E8%B0%83%E7%94%A8%E6%A8%A1%E5%9E%8B%E6%97%B6%E7%9A%84%E5%B9%B6%E5%8F%91%E9%99%90%E5%88%B6%E6%98%AF%E5%A4%9A%E5%B0%91%E6%98%AF%E5%90%A6%E5%8F%AF%E4%BB%A5%E6%8F%90%E9%AB%98%E8%B4%A6%E5%8F%B7%E7%9A%84%E5%B9%B6%E5%8F%91%E4%B8%8A%E9%99%90 "调用模型时的并发限制是多少？是否可以提高账号的并发上限？的直接链接")
当前阶段，我们没有按照用户设置硬性并发上限。在系统总负载量较高时，基于系统负载和用户短时历史用量的动态限流模型可能会导致用户收到 503 或 429 错误码。
目前暂不支持针对单个账号提高并发上限，感谢您的理解。
### 为什么我感觉 API 返回比网页端慢[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%B8%BA%E4%BB%80%E4%B9%88%E6%88%91%E6%84%9F%E8%A7%89-api-%E8%BF%94%E5%9B%9E%E6%AF%94%E7%BD%91%E9%A1%B5%E7%AB%AF%E6%85%A2 "为什么我感觉 API 返回比网页端慢的直接链接")
网页端默认使用流式输出（stream=true），即模型每输出一个字符，都会增量地显示在前端。
API 默认使用非流式输出（stream=false），即模型在所有内容输出完后，才会返回给用户。您可以通过开启 API 的 stream 模式来提升交互性。
### 为什么调用 API 时，持续返回空行？[​](https://api-docs.deepseek.com/zh-cn/faq#%E4%B8%BA%E4%BB%80%E4%B9%88%E8%B0%83%E7%94%A8-api-%E6%97%B6%E6%8C%81%E7%BB%AD%E8%BF%94%E5%9B%9E%E7%A9%BA%E8%A1%8C "为什么调用 API 时，持续返回空行？的直接链接")
为了保持 TCP 连接不会因超时中断，我们会在请求等待调度过程中，持续返回空行（非流式请求）或 SSE keep-alive 注释（`: keep-alive`，流式请求）。如果您在自己解析 HTTP 响应，请注意处理这些空行或注释。
### 是否支持 LangChain？[​](https://api-docs.deepseek.com/zh-cn/faq#%E6%98%AF%E5%90%A6%E6%94%AF%E6%8C%81-langchain "是否支持 LangChain？的直接链接")
支持。LangChain 支持 OpenAI API 接口，而 DeepSeek API 接口与 OpenAI 兼容。您可以下载以下代码文件并替换代码中的 API Key，实现在 LangChain 中调用 DeepSeek API。
